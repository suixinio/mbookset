# BookSet



# so爬虫使用

1、爬取questions
```
p 是页数
https://bookset.io/so/questions?p=1000
```

2、爬取answers
```
s 是单次获取多少个问题
l 是循环获取多少次

https://www.bookset.io/so/answers?s=10&l=100
```

3、翻译so
```
flag 
    q 是翻译问题 
    a 是翻译答案

q 存在 s 存在  则翻译有answer的问题
a 存在  ， 翻译所有的回答

s 是每次的条数

https://www.bookset.io/so/translate?flag=qa
```
